Запуск проекта
Программа будет непрерывно слушать пользователя и выполнять доступные функции, пока не будет получена команда: "конец работы".

Основные возможности:
Включение любой песни:
Команда: "Включи песню" + название песни

Открытие YouTube:
Команда: "Открой ютуб"

Создание скриншота:
Команда: "Сделай скриншот"

Выключение компьютера:
Команда: "Выключи компьютер"

Открытие приложений:
Команда: "Открой" + название приложения

Перезагрузка компьютера:
Команда: "Перезагрузи компьютер"

Работа с папками (в пределах директории проекта):
Создание папки:
Команда: "Создай папку" + имя

Удаление папки:
Команда: "Удали папку" + имя

Открытие папки:
Команда: "Открой папку" + имя

Переименование папки:
Команда: "Измени имя папки" + текущее имя + "на" + новое имя

Новые улучшения:
Озвучка:
Программа теперь озвучивает свои ответы.

Управление системой:
Смена языка:
Команда: "Смени язык"

Управление громкостью:
Команды:
"Уменьши громкость на [число]"
"Увеличь громкость на [число]"

Управление окнами:
Команды:
"Закрой окно"
"Открой на весь экран"
"Сверни все окна"
"Сверни окно"

Проверка активности:
Команда: "Джарвис ты тут"
Ответ: программа подтверждает своё присутствие

Работа с браузером (после открытия, например, Chrome):
Поиск:
Команда: "Поиск" (далее ввод запроса)

Закрытие вкладки:
Команда: "Закрыть"

Создание новой вкладки:
Команда: "Добавить"

Интеграция с Telegram:
Открыть приложение Telegram:
Команда: "Открой Telegram"

Отправка сообщения:
Команда:
"Напиши пользователю [имя]":
программа отвечает
пользователь диктует сообщение
сообщение отправляется
