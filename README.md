Программа будет без остановки вас слушать и выполнять функции которые она умеет, пока вы не скажете: "конец работы"
Программа умеет:
  1. Включать любую песню, достаточно сказать: "Включи песню" а затем название самой песни
  2. Открывать ютуб для этого достаточно сказать: "Открой ютуб"
  3. Делать скриншот для этого достаточно сказать: "Сделай скриншот"
  4. Выключать компьютер для этого достаточно сказать: "выключи компьютер"
  5. Открывать любое приложение которое есть на компьютере для этого достаточно сказать: "Открой" а затем название самого приложения
  6. Перезагружать компьютер для этого достаточно сказать: "перезагрузи компьютер"
  7. А также манипулировать с папками, но это работает только внутри той папки в которой находится код:
  8.   создать папку - "создай папку" а затем название которое вы хотите дать этой папке
  9.   удалить папку - "удали папку" а затем название папки которую вы хотите удалить
  10.  открыть папку - "открой папку" а затем название папки которую вы хотите открыть
  11.  изменить имя папки - "измени имя папки" а затем название самой папки, "на", а затем новое название которое вы хотите дать папке
